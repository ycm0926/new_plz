**문제 설명**  
N!에서 뒤에서부터 처음 0이 아닌 숫자가 나올 때까지 0의 개수를 구하는 프로그램을 작성하시오.

**입력**  
첫째 줄에 N이 주어진다. (0 ≤ N ≤ 500)

**출력**  
첫째 줄에 구한 0의 개수를 출력한다.

```python
n = int(input())
ans = 1
cnt = 0
for i in range(1,n+1): # 팩토리얼 코드
    ans *= i

lst = list(str(ans))
for j in lst[-1::-1]: # 0 개수 확인
    if j == '0':
        cnt +=1
    if cnt != 0 and j != '0':
        break

print(cnt)
```
